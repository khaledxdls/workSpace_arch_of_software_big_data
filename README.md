# My Express App

A simple web application built with Node.js and Express.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/khaledxdls/workSpace_arch_of_software_big_data.git
   cd "web app"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the server:

```bash
npm start
```

For development with auto-reload:

```bash
npm run dev
```

The application will be available at: **http://127.0.0.1:5000/**

## Features

- Simple form input
- Echoes user input back to the page
